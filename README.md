Gearbox Specifications:   
Stages: 2    
Gear Ratio: 25:1 (5:1 × 5:1)    
Module: 1    
Sun Gear Teeth: 12   
Planet Gear Teeth: 18   
Ring Gear Teeth: 48  


Recommended Motors:  
NEMA 17 stepper motor  
You can redesign the bottomshell.step to fit your motor size. 

--------------------------------------
3D Printing Settings:  
Layer Height: 0.2 mm    
Infill: 15-30% (honestly 15% is enough but no harm in trying 30%)  
Material: PLA but PETG is recommended    
Supports: tree-default-(angle 30)-remove all overhangs

<br>
6XM4 nuts

--------------------------------------
Assembly:
1. Install all M2.5 screws int the pre drilled slots on the motor.
2. Place sun gear on motor shaft.
3. Line up the Bottom shell with M2.5 screws installed on the motor and push it downwards until you hear or feel a click.
4. Insert planet gears into First gear holder.
5. Install First output ring on top of the first gear holder and line up there holes and screw it using 4 of the (M4X8) screws.
6. Insert the fully assembaled first gear holder inside the bottom shell.
7. Insert all M4x20 screws in a flipped postion to the side holes found on the sides of the bottom shell.
8. Install 2 washers on each M4X20 screws. 
9. Insert planet gears into Second gear holder.
10. Install Second output ring on top of the Second gear holder and line up there holes and screw it using the reaming 4 of the(M4X8) screws.
11. Insert the fully assembaled Second gear holder on top of the First output ring.
12. Rotate the Second set of planetary gears to match the rotation of the first planetary gear set.
13. Then aling the Top shell with the Bottom shell using the screws we installed on step 7 for alignment. 
14. Install nuts on each side screw to screw everything toghter.
15. Done!!
